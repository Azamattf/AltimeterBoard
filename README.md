# AltimeterBoard

This is an ongoing project aiming to build a Teensy-based barometric altimeter that displays altitude on a display.

Organization: Techical University of Munich

Lab: Spacecraft Electronics

Team: 3


### For Contributors

After cloning the repo, open the [src](https://github.com/Azamattf/AltimeterBoard/tree/542f54cf3dc0a87381f6aebc43887599fe348976/src) folder using **Platformio**.
